# 2023-2-BeachTennisCoordiMate

Repositório de projeto da disciplina de REQ-T3.

## ❗ Requisitos

- Docker
- Node
- npm
- Prisma ORM

## 🛞 Como executar/rodar

1. **Clone o repositório**

```
git clone https://github.com/mdsreq-fga-unb/2023-2-BeachTennisCoordiMate.git
```

2. **Backend**

```
cd backend
docker compose build
docker compose up
```

3. **Frontend**

```
cd frontend
npm i
npm run start
```
